<TabItem label='Question'>
:::note[Difficulty: **Easy** - Category: **Structural Analysis**]
A cantilever beam length $L = 8~\text{ft}$ carries a uniform service load $w = 0.8~\text{k/ft}$. Compute the shear at the fixed support in kips.
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
Total load on cantilever: $wL = 0.8\times8 = 6.4~\text{kips}$.
Shear at fixed support equals total load (upward reaction balancing downward load), so **shear = 6.4 kips**.
:::
</TabItem>
