<TabItem label='Question'>
:::note[Difficulty: **Easy** - Category: **Prestressed Concrete**]
Using the empirical rule $l_t\approx12 d_b$, compute approximate transfer lengths for tendon (or bar) diameters: $d_b = 3/8\"$, $1/2\"$, $5/8\"$, and $3/4\"$ (inches).
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
Apply $l_t\approx 12 d_b$:

- $d_b=3/8=0.375$ in → $l_t\approx 12(0.375)=4.50$ in.  
- $d_b=1/2=0.500$ in → $l_t\approx 12(0.500)=6.00$ in.  
- $d_b=5/8=0.625$ in → $l_t\approx 12(0.625)=7.50$ in.  
- $d_b=3/4=0.750$ in → $l_t\approx 12(0.750)=9.00$ in.

**Answer:** Approximate transfer lengths: 4.5 in, 6.0 in, 7.5 in, 9.0 in respectively (use code models for final).
:::
</TabItem>
