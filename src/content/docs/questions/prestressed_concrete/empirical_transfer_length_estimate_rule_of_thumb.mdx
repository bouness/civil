<TabItem label='Question'>
:::note[Difficulty: **Easy** - Category: **Prestressed Concrete**]
Using a common rule-of-thumb $l_t\approx 12d_b$ (in inches) estimate the transfer length for a prestressing element of nominal diameter $d_b = 0.60~\text{in}$ (approx. 3/4" strand equivalent).
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
Using the empirical rule:

$$
l_t \approx 12 d_b = 12(0.60) = 7.20~\text{in}.
$$

**Answer:** approximate transfer length $l_t \approx 7.2~\text{in}$.  (Use code or refined bond models for final design.)
:::
</TabItem>
