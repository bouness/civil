<TabItem label='Question'>
:::note[Difficulty: **Easy** - Category: **Serviceability**]
A simply supported floor beam span is $L = 18~\text{ft} = 216~\text{in}$. Code limits live-load deflection to $L/360$. What is the maximum allowable live-load deflection (in inches)?
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
**Compute limit:**

$$
\delta_{max} = \frac{L}{360} = \frac{216}{360}
$$


$$
\delta_{max} = 0.6~\text{in}
$$

Thus, **maximum allowable live-load deflection = 0.6 in.**
:::
</TabItem>
