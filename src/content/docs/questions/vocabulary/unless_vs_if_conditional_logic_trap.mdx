<TabItem label='Question'>
:::caution[Difficulty: **Medium** - Category: **Vocabulary**]
Which logical condition is equivalent to the statement **"Do X unless Y"** ?

A) If not Y, then do X.  
B) If Y, then do X.  
C) Do X only if Y.
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
Correct answer: **A**.

**Why:** The phrase **"Do X unless Y"** means perform X in all cases except when Y is true. In formal logic: "Do X unless Y" ↔ "If not Y, then do X". Common exam trap: misreading it as "do X only if Y" which is the opposite (that would require Y).
:::
</TabItem>
