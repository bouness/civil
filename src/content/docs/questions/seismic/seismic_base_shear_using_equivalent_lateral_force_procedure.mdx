<TabItem label='Question'>
:::danger[Difficulty: **Hard** - Category: **Seismic**]
A 3-story regular building has total seismic weight $W = 600~\text{kips}$. Use an equivalent lateral force procedure with response modification factor $R = 6$, importance factor $I_e = 1.0$, and spectral acceleration parameter $S_{DS} = 0.5g$. Compute the design base shear $V = \frac{S_{DS}W}{R/I_e}$ (simplified expression).
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
**Compute denominator first:**

$$
\frac{R}{I_e} = \frac{6}{1.0} = 6
$$

**Design base shear (use $S_{DS}W/(R/I_e)$):**

$$
V = \frac{S_{DS} W}{R/I_e} = \frac{0.5(600)}{6}
$$


$$
V = \frac{300}{6} = 50~\text{kips}
$$

Thus, **design base shear $V = 50~\text{kips}$.**
:::
</TabItem>
