<TabItem label='Question'>
:::caution[Difficulty: **Medium** - Category: **Concrete**]
Per common code guidance, maximum spacing of temperature-and-shrinkage reinforcement in slabs should not exceed $3h$ (where $h$ is slab thickness). If slab thickness $h = 6~\text{in}$, what is the maximum bar spacing in inches and feet?
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
Maximum spacing = $3h = 3\times6~\text{in} = 18~\text{in}$.
Convert to feet: $18~\text{in} = 1.5~\text{ft}$.
Thus maximum bar spacing = **18 in (1.5 ft)**.
:::
</TabItem>
