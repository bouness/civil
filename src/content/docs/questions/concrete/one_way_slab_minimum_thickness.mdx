<TabItem label='Question'>
:::note[Difficulty: **Easy** - Category: **Concrete**]
Determine the minimum slab thickness for a **simply supported one-way slab** with span $L = 15~\text{ft}$ using ACI 318 deflection control limits.
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
ACI 318 gives $h_{min} = L/20$ for simply supported one-way slabs.

$$
h = \frac{15 \times 12}{20} = 9~\text{in}
$$

Thus, minimum thickness = **9 inches**.
:::
</TabItem>
