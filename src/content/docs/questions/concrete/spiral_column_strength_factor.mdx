<TabItem label='Question'>
:::tip[Difficulty: **Conceptual** - Category: **Concrete**]
Compare the strength reduction factor $\phi$ used for spiral columns versus tied columns in ACI 318. Explain the reason for the difference.
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
**Answer:**
- For **tied columns**, $\phi = 0.65$.
- For **spiral columns**, $\phi = 0.70$.

**Reason:** Spiral reinforcement provides **better confinement** of the core concrete, delaying spalling and allowing higher ductility and strength after peak load. Hence, ACI allows a higher $\phi$ value for spiral columns.
:::
</TabItem>
