<TabItem label='Question'>
:::danger[Difficulty: **Hard** - Category: **Beams**]
A W21x44 steel beam is made composite with a 5-inch concrete slab ($f'_c = 4000~\text{psi}$) on 3-inch metal deck. Beam spacing = 8 ft, span = 30 ft. The service live load is 100 psf. Calculate the live load deflection using the effective moment of inertia method. ($E_s = 29,000~\text{ksi}$, $n = E_s/E_c = 8$)
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
**Composite Beam Deflection**

**Step 1: Calculate transformed section properties**
Effective flange width: $b_{eff} = \min(\frac{L}{4}, \text{spacing}) = \min(\frac{30}{4}, 8) = 7.5~\text{ft} = 90~\text{in}$

Concrete area transformed: $\frac{b_{eff}}{n} \times t_s = \frac{90}{8} \times 5 = 11.25 \times 5 = 56.25~\text{in}^2$

**Step 2: Locate elastic neutral axis**
W21x44 properties: $A_s = 13.0~\text{in}^2$, $I_s = 843~\text{in}^4$, $d = 20.66~\text{in}$

Take moments about top of steel:
$\bar{y} = \frac{A_s \times (\frac{d}{2} + 5) + A_c \times 2.5}{A_s + A_c} = \frac{13.0 \times (10.33 + 5) + 56.25 \times 2.5}{13.0 + 56.25}$

$\bar{y} = \frac{13.0 \times 15.33 + 140.6}{69.25} = \frac{199.3 + 140.6}{69.25} = \frac{339.9}{69.25} = 4.91~\text{in}$ from top of steel

**Step 3: Calculate transformed moment of inertia**
$I_{tr} = I_s + A_s (\text{steel lever arm})^2 + \frac{I_c}{n} + A_c (\text{concrete lever arm})^2$

Steel lever arm = $(\frac{d}{2} + 5) - \bar{y} = 15.33 - 4.91 = 10.42~\text{in}$
Concrete lever arm = $\bar{y} - 2.5 = 4.91 - 2.5 = 2.41~\text{in}$

$I_c = \frac{b t_s^3}{12} = \frac{90 \times 5^3}{12} = \frac{90 \times 125}{12} = 937.5~\text{in}^4$

$I_{tr} = 843 + 13.0 \times (10.42)^2 + \frac{937.5}{8} + 56.25 \times (2.41)^2$
$I_{tr} = 843 + 13.0 \times 108.6 + 117.2 + 56.25 \times 5.81$
$I_{tr} = 843 + 1412 + 117.2 + 327 = 2699~\text{in}^4$

**Step 4: Calculate deflection**
Live load: $w_L = 100 \times 8 = 800~\text{plf} = 66.67~\text{lb/in}$

$\Delta_L = \frac{5wL^4}{384EI} = \frac{5 \times 66.67 \times (360)^4}{384 \times 29,000,000 \times 2699}$

$\Delta_L = \frac{5 \times 66.67 \times 1.679\times10^{10}}{384 \times 29\times10^6 \times 2699} = \frac{5.59\times10^{12}}{2.99\times10^{13}} = 0.187~\text{in}$

**Live load deflection: 0.19 inches ($L/1895$, well within $L/360$ requirement)**
