<TabItem label='Question'>
:::caution[Difficulty: **Medium** - Category: **Beams**]
A W21x62 beam ($r_y = 1.77~\text{in}$) with $F_y = 50~\text{ksi}$ spans 30 ft. Determine the maximum unbraced length for which no lateral bracing is required (i.e., $L_p$) and the required bracing spacing to develop the plastic moment capacity.
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
**Beam Bracing Requirements**

**Step 1: Calculate $L_p$ for full plastic capacity**
$L_p = 1.76r_y\sqrt{\frac{E}{F_y}}$

$L_p = 1.76\times1.77\times\sqrt{\frac{29,000}{50}} = 3.115\times\sqrt{580} = 3.115\times24.08 = 75.0~\text{in} = 6.25~\text{ft}$

**Step 2: Calculate $L_r$ for elastic LTB**
For W21x62: $h_0 = d - t_f = 20.99 - 0.615 = 20.38~\text{in}$
$c = 1.0$ (for doubly symmetric I-shape)

$L_r = 1.95r_{ts}\frac{E}{0.7F_y}\sqrt{\frac{Jc}{S_x h_0} + \sqrt{(\frac{Jc}{S_x h_0})^2 + 6.76(\frac{0.7F_y}{E})^2}}$

Using approximate value from steel manual: $L_r \approx 16.9~\text{ft}$

**Step 3: Determine bracing requirements**
- If unbraced length ≤ 6.25 ft: No bracing needed for full plastic capacity
- If 6.25 ft < unbraced length ≤ 16.9 ft: Inelastic LTB, reduced capacity
- If unbraced length > 16.9 ft: Elastic LTB, significant reduction

**Step 4: Required bracing for plastic design**
For plastic design, brace points must be spaced at $L_p$ or less:

Number of braces = $\frac{30}{6.25} - 1 = 4.8 - 1 = 3.8$

Use 4 braces at 6 ft spacing

**Maximum unbraced length for full plastic capacity: 6.25 ft**
**Required bracing: 4 points at 6 ft spacing for 30 ft span**
:::
