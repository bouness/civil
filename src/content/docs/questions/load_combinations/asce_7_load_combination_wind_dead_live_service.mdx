<TabItem label='Question'>
:::note[Difficulty: **Easy** - Category: **Load Combinations**]
For service-level load combination $D + L + W$ (unfactored), compute the service uniform load per ft if $D = 1.8~\text{k/ft}$, $L = 1.5~\text{k/ft}$, and wind $W$ (horizontal) does not act as vertical uniform load (so take $W = 0$).
:::
</TabItem>

<TabItem label='Solution'>
:::tip[Solution]
Service combination is simply sum: $w_s = D + L + W = 1.8 + 1.5 + 0 = 3.3~\text{k/ft}$.
Thus service uniform load = **3.3 k/ft**.
:::
</TabItem>
